# service_provenance

This service is using the FastAPI python framework.

## Installation


### Install requirements

`pip install -r requirements.txt`

Run the service with uvicorn
`python run.py`

### Docker

*docker-compose*

`docker-compose build`
`docker-compose up`

### Unit Test
*directly run command below
`pytest`
